## 详细描述

### 中文详细描述

小红书自动运营助手是一款专为小红书内容创作者和运营人员设计的 Chrome 扩展。

**主要功能：**

📊 **作者数据获取**
- 一键抓取小红书作者的粉丝数、获赞数、收藏数
- 自动获取 TOP 10 笔记的标题、点赞、评论、收藏等核心数据
- 数据在侧边栏直观展示，支持快速对比分析

✍️ **AI 智能仿写**
- 基于知识库模板，AI 一键生成高质量仿写笔记
- 生成结果自动拆分标题、正文、话题标签，方便直接使用
- 支持生成前实时流式预览，生成后可自由编辑修改
- 点击"保存草稿"自动填入小红书创作者中心，无需手动复制粘贴

💬 **互动管理（开发中）**
- 支持对指定笔记发表评论或回复
- 适合批量互动、涨粉运营等场景

🔐 **登录状态检测**
- 自动检测小红书登录状态，未登录时一键跳转登录页
- 不存储任何账号密码，完全依赖浏览器原生 Cookie

**隐私与安全：**
本扩展所有操作均在本地浏览器内完成，不会将您的 Cookie、个人信息或笔记内容上传至任何服务器。AI 仿写仅发送匿名的文本模板，不含个人信息。

**使用前提：**
需已在浏览器中登录小红书网页版（www.xiaohongshu.com）。

### English Detailed Description

XHS Auto-Operations Assistant is a Chrome extension built for Xiaohongshu (RED/小红书) content creators and social media operators.

**Key Features:**

📊 **Author Analytics**
- One-click fetch of follower count, likes, and collected stats for any XHS author
- Auto-retrieve top 10 notes with title, likes, comments, and saves
- Beautifully displayed in the side panel for quick competitive analysis

✍️ **AI Note Rewriting**
- Generate high-quality rewritten notes with one click using AI and a smart knowledge base template
- Output is automatically parsed into title, body, and hashtags
- Real-time streaming preview during generation; fully editable before saving
- Click "Save Draft" to auto-fill the Xiaohongshu Creator Center — no copy-paste needed

💬 **Engagement Tools (in development)**
- Post comments or replies to specific notes automatically
- Useful for community engagement and growth campaigns

🔐 **Login Detection**
- Detects your Xiaohongshu login status instantly
- No passwords stored — uses your browser's native cookies only

**Privacy & Security:**
All operations run locally in your browser. No cookies, personal data, or note content is sent to any external server. The AI rewrite feature only transmits anonymous template text.

**Prerequisites:**
You must be logged into Xiaohongshu (www.xiaohongshu.com) in your browser before using this extension.
